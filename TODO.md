## 2. 移動系（最重要）

### 行・列ジャンプ

| キー | 処理     |
| -- | ------ |
| H  | 画面上端の行 |
| M  | 画面中央の行 |
| L  | 画面下端の行 |

## 4. 列操作（TSVエディタの肝）

### 列単位操作

| キー   | 処理       |
| ---- | -------- |
| `dc` | 現在列を削除   |
| `yc` | 現在列をコピー  |

※ `c` = column の略（Vimらしい拡張）
