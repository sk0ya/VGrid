## 2. 移動系（最重要）

### 行・列ジャンプ

| キー | 処理     |
| -- | ------ |
| $  | 行の末尾列  |
| H  | 画面上端の行 |
| M  | 画面中央の行 |
| L  | 画面下端の行 |

### TSV特有の移動（おすすめ）

| キー | 処理            |
| -- | ------------- |
| }  | 次の空行（レコード区切り） |
| {  | 前の空行          |

## 4. 列操作（TSVエディタの肝）

### 列単位操作

| キー   | 処理       |
| ---- | -------- |
| `dc` | 現在列を削除   |
| `yc` | 現在列をコピー  |
| `pc` | 右に列を貼り付け |
| `Pc` | 左に列を貼り付け |
| `>>` | 列を右へ移動   |
| `<<` | 列を左へ移動   |

※ `c` = column の略（Vimらしい拡張）
